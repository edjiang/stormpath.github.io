Stormpath.SDK.Oauth Namespace
The Stormpath.SDK.Oauth namespace provides classes that are used to implement token-based authentication.
Classes
  ClassDescription
JwtLocalValidationOptions
Represents options used for local validation of JWTs.
OauthRequests
Static entry point for working with OAuth 2.0 requests.
Interfaces
  InterfaceDescription
IAccessToken
Represents an OAuth 2.0 access token created in Stormpath.
IIdSiteTokenAuthenticationRequest
Represents a request for Stormpath to create a new OAuth 2.0 Access Token from an existing ID Site response.
IIdSiteTokenAuthenticationRequestBuilder
Builder pattern used to construct IIdSiteTokenAuthenticationRequest instances.
IIdSiteTokenAuthenticator
Represents a an OAuth 2.0 Authenticator used to validate and exchange an ID Site response for an access token.
IJwtAuthenticationRequest
Represents a JSON Web Token validation request, which can be executed either locally or against the Stormpath API.
IJwtAuthenticationRequestBuilder
Builder pattern used to construct IJwtAuthenticationRequest instances.
IJwtAuthenticator
Represents a an OAuth 2.0 Authenticator used to authenticate a JSON Web Token against Stormpath.
IOauthAuthenticationRequestBuilderTRequest
Builder pattern used to construct Authentication Request instances.
IOauthAuthenticatorTRequest, TResult
Represents the operations that every OAuth 2.0 Authenticator type must support.
IOauthGrantAuthenticationResult
Represents the result of a successful Password or Refresh Grant request.
IOauthGrantRequest
Represents an OAuth 2.0 Grant Request.
IOauthPolicy
An OauthPolicy is used to configure different aspects of the OAuth tokens associated with an Application.
IPasswordGrantAuthenticator
Represents a Password Grant-specific OAuth 2.0 Authenticator used to authenticate an account and exchange its credentials for a valid OAuth 2.0 token.
IPasswordGrantRequest
Represents a request for Stormpath to authenticate an Account and exchange its credentials for a valid OAuth 2.0 access token.
IPasswordGrantRequestBuilder
Builder pattern used to construct IPasswordGrantRequestBuilder instances.
IRefreshGrantAuthenticator
Represents a Refresh Grant-specific OAuth 2.0 Authenticator used to refresh an OAuth 2.0 token created in Stormpath.
IRefreshGrantRequest
Represents a request for Stormpath to create a new OAuth 2.0 Access Token for a previously authenticated account.
IRefreshGrantRequestBuilder
Builder pattern used to construct IRefreshGrantRequest instances.
IRefreshToken
Represents an OAuth 2.0 refresh token created in Stormpath.
