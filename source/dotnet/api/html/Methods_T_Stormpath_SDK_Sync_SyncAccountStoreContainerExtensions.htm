AddAccountStoreTMapping(IAccountStoreContainerTMapping, IAccountStore)
Synchronously adds a new Account Store to the Application or Organization and appends the resulting IAccountStoreMapping to the end of the AccountStoreMapping list.
AddAccountStoreTMapping(IAccountStoreContainerTMapping, String)
Synchronously adds a new Account Store to the Application or Organization. The given string can either be an href or a name of a Directory or Group belonging to the current Tenant.
CreateAccountStoreMappingTMapping
Synchronously creates a new IAccountStoreMapping, allowing the associated Account Store to be used a source of accounts that may log in to the Application or Organization.
GetDefaultAccountStoreTMapping
Synchronously gets the Account Store (either a Group or Directory) used to persist new Accounts, or if no default Account Store has been designated.
GetDefaultGroupStoreTMapping
Synchronously gets the Account Store used to persist new Groups, or if no default Account Store has been designated.
SetDefaultAccountStoreTMapping
Synchronously sets the Account Store (either a Group or a Directory) used to persist new Accounts.
SetDefaultGroupStoreTMapping
Synchronously sets the Account Store (a Directory) used to persist new Groups.
