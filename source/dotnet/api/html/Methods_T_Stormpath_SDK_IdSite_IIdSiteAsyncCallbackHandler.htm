IIdSiteAsyncCallbackHandler Methods

The IIdSiteAsyncCallbackHandler type exposes the following members.

Methods
  NameDescription
GetAccountResultAsync
Processes the request and returns an IAccountResult object that reflects the account that logged in or registered.
SetNonceStore
Sets a INonceStore that will retain ID Site message identifiers as cryptographic nonces, eliminating the possibility of a replay attack. This ensures any ID Site message cannot be intercepted and used again later.
SetResultListener(IIdSiteAsyncResultListener)
Sets the IIdSiteAsyncResultListener that will be notified about the actual operation of the ID Site invocation: registration, authentication, or logout.
SetResultListener(FuncIAccountResult, CancellationToken, Task, FuncIAccountResult, CancellationToken, Task, FuncIAccountResult, CancellationToken, Task)
Sets the IIdSiteAsyncResultListener that will be notified about the actual operation of the ID Site invocation: registration, authentication, or logout. This overload constructs an inline IIdSiteAsyncResultListener based on the delegate parameters.
