Stormpath.SDK.AccountStore Namespace
The Stormpath.SDK.AccountStore namespace provides classes that are used to interact with AccountStore resources (Directories and Groups), and AccountStoreMapping resources.
Interfaces
  InterfaceDescription
IAccountStore
An abstract representation of a Directory or Group.
IAccountStoreContainerTMapping
Represents Resources that are capable of storing AccountStores, such as Applications and Organizations.
IAccountStoreMapping Obsolete.
Represents the assignment of an Account Store AccountStore (either a Group or Directory) to an Application.
IAccountStoreMappingT
Represents the assignment of an Account Store AccountStore (either a Group or Directory) to an Application.
