IApplicationAuthenticateAccountAsync Method (ActionUsernamePasswordRequestBuilder, CancellationToken)
Authenticates an account's submitted principals and credentials (e.g. username and password) against the specified account store. If the account does not exist in the account store, the authentication attempt will fail.
