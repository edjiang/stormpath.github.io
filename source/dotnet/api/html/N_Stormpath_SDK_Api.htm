Stormpath.SDK.Api Namespace
The Stormpath.SDK.Api namespace provides classes that interact with Stormpath API Key instances.
Classes
  ClassDescription
ApiKeyStatus
Represents the status (usability) of an API Key instance.
ClientApiKeys
Static entry point for working with Client API Key objects.
Interfaces
  InterfaceDescription
IApiKey
An API Key is a secure random username/password pair attributed to an Account that can be used by the account to make secure requests to an API service.
IClientApiKey
Represents a Stormpath customer's API-specific ID and secret. All Stormpath REST invocations must be authenticated with a Client API Key.
IClientApiKeyBuilder
A Builder design pattern used to construct Client API Key instances.
