AddAccountStoreAsync(String, CancellationToken)
Adds a new Account Store to this Application. The given string can either be an href or a name of a Directory or Group belonging to the current Tenant.
AddAccountStoreAsync(IAccountStore, CancellationToken)
Adds a new Account Store to the Application or Organization and appends the resulting Account Store Mapping to the end of the AccountStoreMapping list.
AddAccountStoreAsyncTSource(FuncIAsyncQueryableTSource, IAsyncQueryableTSource, CancellationToken)
Adds a resource of type TSource as a new Account Store to this Application or Organization. The provided query must match a single TSource in the current Tenant. If no compatible resource matches the query, this method will return .
CreateAccountStoreMappingAsync
Creates a new Account Store Mapping, allowing the associated Account Store to be used a source of accounts that may log in to the Application or Organization.
GetAccountStoreMappings
Gets a queryable list of all Account Store Mappings accessible to the resource.
GetDefaultAccountStoreAsync
Gets the Account Store (either a Group or Directory) used to persist new Accounts, or if no default Account Store has been designated.
GetDefaultGroupStoreAsync
Gets the Account Store used to persist new Groups, or if no default Account Store has been designated.
