IIdSiteSyncCallbackHandler Methods

The IIdSiteSyncCallbackHandler type exposes the following members.

Methods
  NameDescription
GetAccountResult
Synchronously processes the request and returns an IAccountResult object that reflects the account that logged in or registered.
SetNonceStore
Sets a INonceStore that will retain ID Site message identifiers as cryptographic nonces, eliminating the possibility of a replay attack. This ensures any ID Site message cannot be intercepted and used again later.
SetResultListener(IIdSiteSyncResultListener)
Sets the IIdSiteSyncResultListener that will be notified about the actual operation of the ID Site invocation: registration, authentication, or logout.
SetResultListener(ActionIAccountResult, ActionIAccountResult, ActionIAccountResult)
Sets the IIdSiteSyncResultListener that will be notified about the actual operation of the ID Site invocation: registration, authentication, or logout. This overload constructs an inline IIdSiteSyncResultListener based on the delegate parameters.
