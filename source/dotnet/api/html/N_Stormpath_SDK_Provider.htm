Stormpath.SDK.Provider Namespace
The Stormpath.SDK.Provider namespace provides classes that are used to interact with Stormpath Providers.

IProvider
A provider resource holds specific information needed to work with Provider-based Directories (e.g, Google and Facebook).
IProviderAccountRequest
Represents an attempt to get or create a Provider-based Account record in Stormpath.
IProviderAccountRequestBuilderT
A builder to construct IProviderAccountRequest instances.
IProviderAccountResult
Represents the result of the attempt to access the Provider's account. Stormpath maps common fields of the Provider user to the Account Resource.
IProviderData
Resource containing Provider-specific data for the Account. For example, for Google, it contains refreshToken and accessToken.
IProviderFactory
Represents a factory that can construct Provider-specific IProviderRequestFactoryA, P instances.
IProviderRequestFactoryA, P
Interface describing the builder capabilities that Providers implement in Stormpath.
