Stormpath.SDK.Auth Namespace
The Stormpath.SDK.Auth namespace provides classes that represent authentication requests and results.
Classes
  ClassDescription
ApiAuthenticationException
Represents an error that occurs while attempting to authenticate an API Key.
ApiKeyRequestBuilder
Constructs an IAuthenticationRequest given an API Key ID and Secret.
DisabledAccountException
Represents an error that occurs during API Authentication when the API Key is found, but the owning Account is disabled.
DisabledApiKeyException
Represents an error that occurs during API Authentication when the API Key is found, but it is Disabled.
IncorrectCredentialsException
Represents an error that occurs during API Authentication when the API Key is not found.
UsernamePasswordRequestBuilder
Constructs an IAuthenticationRequest given a username and password.
Interfaces
  InterfaceDescription
IAuthenticationRequest
The default implementation of IAuthenticationRequestP, C that uses strings for both principals and credentials.
IAuthenticationRequestP, C
An authentication request represents all necessary information to authenticate a specific account.
IAuthenticationResult
Represents the return value of an Application authentication attempt. The successfully authenticated account may be obtained via GetAccountAsync(CancellationToken).
