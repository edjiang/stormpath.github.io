SetEmail
Sets the account's email address, which must be unique among all other accounts within a Directory.
SetGivenName
Sets the account's given name (aka 'first name' in Western cultures').
SetMiddleName
Sets the account's middle name(s).
SetPassword
Sets (changes) the account's password to the specified raw (plaintext) password. Only call this method if you legitimately want to set the account password directly. When possible, it is advisable to use Stormpath's account password reset email workflow instead.

After calling this method, you must call SaveAsync(CancellationToken) to propagate the change to the Stormpath servers.

SetStatus
Sets the account's status. Accounts that are not Enabled may not login to applications.
SetSurname
Sets the account's surname (aka 'last name' in Western cultures).
SetUsername
Sets the account's username, which must be unique among all other accounts within a Directory. If you do not have need of a username, it is best to set the username to equal the Email.
