Stormpath.SDK.IdSite Namespace
The Stormpath.SDK.IdSite namespace provides classes that are used to interact with ID Site.
Classes
  ClassDescription
IdSiteResultStatus
Represents the possible results of an ID Site invocation.
IdSiteRuntimeException
A sub-class of ResourceException representing an ID Site error.
IdSiteSessionTimeoutException
This exception indicates that the session on ID Site has timed out.
InvalidIdSiteTokenException
This exception indicates that the token is invalid. Reasons for this could be: An expired token, the issued at time (iat) is after the current time, the specified organization name key does not exist in your Stormpath Tenant, or because the specified organization is disabled.
Interfaces
  InterfaceDescription
IAccountResult
Represents the result of an ID Site callback redirect, obtained after the end-user has used the ID Site and has been returned to an application's callbackUri.
IAsynchronousNonceStore
Store INonce values and provides methods to check if a nonce has already been used, and to add it to the INonceStore to ensure that the same nonce cannot be used again.
IIdSiteAsyncCallbackHandler
Handles HTTP replies sent from your ID Site to your application's callbackUri and returns an IAccountResult.
IIdSiteAsyncResultListener
Listener interface to get asynchronous notifications about effective operations of the ID Site invocation (registration, authentication, or logout).
IIdSiteSyncCallbackHandler
Handles HTTP replies sent from your ID Site to your application's callbackUri and returns an IAccountResult.
IIdSiteSyncResultListener
Listener interface to get synchronous notifications about effective operations of the ID Site invocation (registration, authentication, or logout).
IIdSiteUrlBuilder
Used to construct a URL you can use to redirect your application users to a hosted login/registration/forgot-password site - what Stormpath calls an 'Identity Site' (or 'ID Site' for short) - for performing common user identity functionality. When the user is done (logging in, registering, etc), they will be redirected back to a callbackUri of your choice.
INonce
A cryptographic nonce representation for values that cannot be used more than once.
INonceStore
Store INonce values and provides methods to check if a nonce has already been used, and to add it to the INonceStore to ensure that the same nonce cannot be used again.
ISynchronousNonceStore
Store INonce values and provides methods to check if a nonce has already been used, and to add it to the INonceStore to ensure that the same nonce cannot be used again.
