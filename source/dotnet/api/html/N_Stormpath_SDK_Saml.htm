Stormpath.SDK.Saml Namespace
The Stormpath.SDK.Saml namespace provides classes that are used to interact with SAML providers.
Classes
  ClassDescription
SamlException
Represents a SAML-related error.
SamlResultStatus
Represents the possible results of a SAML invocation.
Interfaces
  InterfaceDescription
ISamlAccountResult
Represents the result of a SAML Identity Provider redirect, obtained after the end-user has interacted with the external site and has been returned to an application's callbackUri.
ISamlAsyncCallbackHandler
Handles HTTP replies sent from a SAML Identity Provider to your application's callbackUri and returns an IAccountResult.
ISamlAsyncResultListener
Listener interface to get asynchronous notifications about effective operations of the SAML Identity Provider invocation (authentication or logout).
ISamlIdpUrlBuilder
Used to construct a URL you can use to redirect application users to a SAML authentication site (Identity Provider or IdP) for performing common user identity functionality. When users are done, they will be redirected back to a callbackUri of your choice.
ISamlPolicy
Represents a SAML Policy associated with an Application.
ISamlServiceProvider
Represents a SAML Service Provider associated with a SAML Policy.
ISamlSyncCallbackHandler
Handles HTTP replies sent from a SAML Identity Provider to your application's callbackUri and returns an IAccountResult.
ISamlSyncResultListener
Listener interface to get synchronous notifications about effective operations of the SAML Identity Provider invocation (authentication or logout).
ISsoInitiationEndpoint
Represents an endpoint that can initiate SAML Single Sign-On.
