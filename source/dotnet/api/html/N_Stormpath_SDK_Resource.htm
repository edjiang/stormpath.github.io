Stormpath.SDK.Resource Namespace
The Stormpath.SDK.Resource namespace provides interfaces that represent abstract Stormpath resources.
Interfaces
  InterfaceDescription
IAuditable
Represents resources that have timestamp fields.
ICreationOptions
Base interface for creation request options objects.
IDeletable
Represents resources that can be deleted.
IExtendable
Represents resources that have CustomData resources that can store arbitrary name/value pairs.
IResource
Represents a resource stored in Stormpath.
IRetrievalOptionsT
Base interface for retrieval request options objects.
ISaveableT
Represents a resource that can be created or modified.
ISaveableWithOptionsT
Represents a resource that can be saved with additional options.
