IIdSiteUrlBuilder Methods

The IIdSiteUrlBuilder type exposes the following members.

Methods
  NameDescription
Build
Builds the fully-qualified URL representing the initial location in the ID Site where the end-user should be redirected.
ForLogout
The built ID Site URL (when invoked) will logout the current user.
SetCallbackUri(String)
Sets the location where the user will be sent when returning from the ID Site. This property is mandatory and must be set. Use NewIdSiteAsyncCallbackHandler(IHttpRequest) to process requests to your callbackUri.
SetCallbackUri(Uri)
Sets the location where the user will be sent when returning from the ID Site. This property is mandatory and must be set. Use NewIdSiteAsyncCallbackHandler(IHttpRequest) to process requests to your callbackUri.
SetOrganizationNameKey
Ensures the ID Site is customized for the Organization with the specified organizationNameKey This is useful for multi-tenant or white label scenarios where you know the user belongs to a specific Organization.
SetPath
Sets the initial path in the ID Site where the user should be sent. If unspecified, this defaults to /, implying that the ID Site's landing/home page is the desired location.
SetShowOrganizationField
Ensures that the ID Site will show the SetOrganizationNameKey(String) field to the end-user in the ID Site user interface.
SetState
Sets application-specific state data that should be retained and made available to your callbackUri when the user returns from the ID Site.
SetUseSubdomain
Ensures that the user will visit ID Site using a subdomain equal to the SetOrganizationNameKey(String) instead of the standard base domain.
