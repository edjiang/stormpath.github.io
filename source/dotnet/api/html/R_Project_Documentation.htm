Namespaces
NamespaceDescription
Stormpath.SDK
The Stormpath.SDK namespace provides classes and extension methods that are used to implement the common behavior of the SDK.
Stormpath.SDK.Account
The Stormpath.SDK.Account namespace provides classes that are used to interact with Account resources.
Stormpath.SDK.AccountStore
The Stormpath.SDK.AccountStore namespace provides classes that are used to interact with AccountStore resources (Directories and Groups), and AccountStoreMapping resources.
Stormpath.SDK.Api
The Stormpath.SDK.Api namespace provides classes that interact with Stormpath API Key instances.
Stormpath.SDK.Application
The Stormpath.SDK.Application namespace provides classes that are used to interact with Application resources.
Stormpath.SDK.Auth
The Stormpath.SDK.Auth namespace provides classes that represent authentication requests and results.
Stormpath.SDK.Cache
The Stormpath.SDK.Cache namespace provides classes that are used to implement client-side caching of API resources.
Stormpath.SDK.Client
The Stormpath.SDK.Client namespace provides classes that are used to interact with the Stormpath REST API.
Stormpath.SDK.CustomData
The Stormpath.SDK.CustomData namespace provides classes that are used to interact with Custom Data resources.
Stormpath.SDK.DataStore
The Stormpath.SDK.DataStore namespace provides classes that handle the underlying communication with the Stormpath REST API.
Stormpath.SDK.Directory
The Stormpath.SDK.Directory namespace provides classes that are used to interact with Directory resources.
Stormpath.SDK.Error
The Stormpath.SDK.Error namespace provides classes that represent Stormpath API errors.
Stormpath.SDK.Group
The Stormpath.SDK.Group namespace provides classes that are used to interact with Group resources.
Stormpath.SDK.Http
The Stormpath.SDK.Http namespace provides classes that represent HTTP requests and responses.
Stormpath.SDK.IdSite
The Stormpath.SDK.IdSite namespace provides classes that are used to interact with ID Site.
Stormpath.SDK.Jwt
The Stormpath.SDK.Jwt namespace provides classes that are used to interact with JSON Web Token (JWT) objects.
Stormpath.SDK.Linq
The Stormpath.SDK.Linq namespace provides classes that are used to implement LINQ to Stormpath.
Stormpath.SDK.Linq.Expandables
The Stormpath.SDK.Linq.Expandables namespace provides interfaces that represent resources and collections that can be expanded.
Stormpath.SDK.Logging
The Stormpath.SDK.Logging namespace provides classes that are used for debug logging.
Stormpath.SDK.Oauth
The Stormpath.SDK.Oauth namespace provides classes that are used to implement token-based authentication.
Stormpath.SDK.Organization
The Stormpath.SDK.Organization namespace provides classes that are used to interact with Organization resources.
Stormpath.SDK.Provider
The Stormpath.SDK.Provider namespace provides classes that are used to interact with Stormpath Providers.
Stormpath.SDK.Resource
The Stormpath.SDK.Resource namespace provides interfaces that represent abstract Stormpath resources.
Stormpath.SDK.Saml
The Stormpath.SDK.Saml namespace provides classes that are used to interact with SAML providers.
Stormpath.SDK.Serialization
The Stormpath.SDK.Serialization namespace provides classes that are used to serialize and deserialize JSON data.
Stormpath.SDK.Shared
The Stormpath.SDK.Shared namespace provides classes that are shared by the SDK.
Stormpath.SDK.Sync
The Stormpath.SDK.Sync namespace provides synchronous access to all SDK API methods.
Stormpath.SDK.Tenant
The Stormpath.SDK.Tenant namespace provides classes that are used to interact with Tenant resources.
