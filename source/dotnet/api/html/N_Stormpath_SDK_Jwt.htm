Stormpath.SDK.Jwt Namespace
The Stormpath.SDK.Jwt namespace provides classes that are used to interact with JSON Web Token (JWT) objects.
Classes
  ClassDescription
ExpiredJwtException
Represents an error indicating that a JSON Web Token has expired.
InvalidJwtException
Represents a JSON Web Token validation error.
Jwts Obsolete.
Utility class for creating IJwtClaimsBuilder instances, used for constructing IJwtClaims instances.
JwtSignatureException
Represents an error indicating that a JSON Web Token has an invalid or mismatched signature.
MalformedJwtException
Represents an error indicating that a JSON Web Token is malformed.
MismatchedClaimException
Represents an error indicating that a JSON Web Token has a missing claim.
MissingClaimException
Represents an error indicating that a JSON Web Token has a missing claim.
PrematureJwtException
Represents an error indicating that a JSON Web Token has been used too early.

IJwtClaims

IJwtClaimsBuilder
Builder pattern to construct IJwtClaims instances.
IJwtParser
